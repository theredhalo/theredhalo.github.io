What is The Red Halo?

The Red Halo is a platform that can help you during an emergency by sending a distress signal to other users of this platform in the nearby area. This project is only getting started and hopefully I'll be able to release a working mobile app soon.

The platform will eventually be way more that what you just read. I have some cool things in mind. For example, the core platform will expose an API for others to use, so that they (eg: Uber) can interact with it (eg: get alerted when a passenger is in need of help, or send us a distress signal from their end which we can push to other users of the platform).

I believe that this project has the potential to help bring down crime rates and make our world a little safer. If you think this is cool, please get involved!
